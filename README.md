# crawler_for_outreach

Prerequisites:
1. Pipenv

To install Pipenv, run:
```shell
pip3 install pipenv
```
Clone this repo using
```shell
git clone https://github.com/Dishank422/crawler_for_outreach.git
```

To setup virtual environment:
```shell
cd crawler_for_outreach
pipenv install
```

To run the program, run:
```shell
pipenv shell
python crawler.py
```
